x = ts(da[, 2], start = 1952, frequency = 1)

判断平稳性

显然这是一个非平稳时间序列，我们首先进行差分

x.1 = diff(x)
plot(x.1, type = "o")

现在序列不再有明显趋势性，为了进一步确定序列的平稳性，可观察差分后序列的 ACF；

acf(x.1)

自相关图只含有显著的短期自相关性，可以认为1阶差分后序列平稳；

定阶

观察其ACF和PACF

par(mfrow = c(1, 2))
acf(x.1)
pacf(x.1)


par(mfrow = c(1, 1))

对比我们认为ACF 1阶截尾，因此采用MA(1)模型，即对原时间序列拟合ARIMA(0,1,1)模型

建模

model1 = arima(x, order = c(0, 1, 1), include.mean = TRUE, method = "CSS")

# model.1 is wrong,why?

注意，上面拟合的模型没有常数项，这是错误的！用?arima查看帮助文档，发现

include.mean  Should the ARMA model include a mean/intercept term? The default is TRUE 
for undifferenced series, and it is ignored for ARIMA models with differencing.

而对差分后的序列进行t检验，发现其均值显著不为零；

t.test(x.1)

## 
##  One Sample t-test
## 
## data:  x.1
## t = 3.333, df = 35, p-value = 0.002038
## alternative hypothesis: true mean is not equal to 0
## 95 percent confidence interval:
##  1.948 8.019
## sample estimates:
## mean of x 
##     4.983

那么如何得到一个正确的模型呢？一种办法是我们对差分后的序列拟合MA(1)模型

model2 = arima(x.1, order = c(0, 0, 1), method = "CSS")
model2

## 
## Call:
## arima(x = x.1, order = c(0, 0, 1), method = "CSS")
## 
## Coefficients:
##         ma1  intercept
##       0.709      4.997
## s.e.  0.173      2.031
## 
## sigma^2 estimated as 53.3:  part log likelihood = -122.7

tsdiag(model2)

模型为 \[ (1-B)x_t=4.996+(1+0.709B)\epsilon_t,~~Var(\epsilon_t)=53.3 \] 另一种办法如下

model3 = arima(x, order = c(0, 1, 1), xreg = 1:length(x), method = "CSS")
model3

## 
## Call:
## arima(x = x, order = c(0, 1, 1), xreg = 1:length(x), method = "CSS")
## 
## Coefficients:
##         ma1  1:length(x)
##       0.709        4.997
## s.e.  0.176        2.059
## 
## sigma^2 estimated as 53.3:  part log likelihood = -119.3

tsdiag(model3)

这相当于拟合如下模型 \[ \nabla (x_t-\mu t)=(1+\theta B)\epsilon_t \] 这两种方法在进行预测时都比较麻烦，因此我建议使用forecast包中的Arima函数 来直接拟合ARIMA(0,1,1)模型

library("forecast")
model4 = Arima(x, order = c(0, 1, 1), include.drift = TRUE, method = "CSS")
model4

## Series: x 
## ARIMA(0,1,1) with drift         
## 
## Coefficients:
##         ma1  drift
##       0.709  4.997
## s.e.  0.176  2.059
## 
## sigma^2 estimated as 53.3:  part log likelihood=-119.3

这里的drift就是差分后序列模型的均值项. \[ \Phi(B)(\nabla x_t-drift)=\Theta(B)\epsilon_t \]

预测

fore = forecast(model4, h = 10)
plot(fore)
