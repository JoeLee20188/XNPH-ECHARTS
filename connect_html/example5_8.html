判断平稳性

显然这是一个非平稳时间序列，我们首先进行差分

x.1 = diff(x)
plot(x.1, type = "o")

现在序列不再有明显趋势性，为了进一步确定序列的平稳性，可观察差分后序列的 ACF；

acf(x.1)

自相关图只含有显著的短期自相关性，可以认为1阶差分后序列平稳；

t检验显示该序列均值不显著;

模型诊断的结果是模型有效，那么是否存在更有效的模型呢？ 回顾该数据PACF只有1阶和4阶有效，我们实际上可以考虑如下 的稀系数模型 \[ (1-\phi_1B-\phi_4B^4)\nabla x_t=\epsilon_t \]

m2 = Arima(x, order = c(4, 1, 0), include.drift = FALSE, fixed = c(NA, 0, 0,

利用稀疏化来优化模型

在实际建模中，我们经常可以使用稀疏模型来优化模型，方法是对建立的模型 进行系数显著性检验，然后反复尝试并删减不显著的系数，但是这个过程需要 小心，要保证模型的有效性;例如对上面的m1做系数显著性检验，发现\( \phi_2,\phi_3 \) 不显著

library("lmtest")
coeftest(m1)
